.. _xml-encoding:

encoding
========

Atributo que especifica a codificação de caracteres usada no texto do documento. Para a :term:`SciELO PS` todos os XMLs devem ser codificados em :term:`UTF-8`.

A especificação do padrão :term:`XML` (`2.8 Prolog and Document Type Declaration <https://www.w3.org/TR/2000/REC-xml-20001006#sec-prolog-dtd>`_) fornece mais informação sobre as características de codificação requeridas para este padrão.


Exemplo:

.. code-block:: xml

    <?xml version="1.0" encoding="utf-8"?>
